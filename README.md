🚀 Prodigy InfoTech Internship Projects

This repository contains multiple machine learning and computer vision projects completed as part of the internship at Prodigy InfoTech. Each task focuses on applying core ML algorithms and real-world problem solving.

📂 Projects Overview
🔹 Task 01: Linear Regression – House Price Prediction
Built a Linear Regression model to predict house prices.
Features used:
Square footage
Number of bedrooms
Number of bathrooms
Goal: Understand relationship between house features and pricing.
🔹 Task 02: K-Means Clustering – Customer Segmentation
Implemented K-Means clustering algorithm.
Grouped retail customers based on purchase behavior.
Helps in:
Customer segmentation
Targeted marketing strategies
🔹 Task 03: SVM – Cat vs Dog Image Classification
Built a Support Vector Machine (SVM) model.
Classified images of cats and dogs using Kaggle dataset.
Focus:
Image preprocessing
Feature extraction
Binary classification
🔹 Task 04: Hand Gesture Recognition System
Developed a computer vision model to detect and classify hand gestures.
Enables gesture-based interaction for HCI systems.
Applications:
Smart devices
Accessibility systems
Touchless control interfaces
