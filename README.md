# 🚀 Prodigy InfoTech Internship Projects

## 📖 Introduction
This repository contains multiple **Machine Learning** and **Computer Vision** projects completed as part of the internship at **Prodigy InfoTech**. Each task demonstrates the application of core ML algorithms to solve real-world problems, including regression, clustering, and image classification.

---

## 📂 Projects Overview

### 🔹 Task 01: Linear Regression – House Price Prediction
Built a Linear Regression model to predict house prices.

**Features used:**
- Square footage  
- Number of bedrooms  
- Number of bathrooms  

**Goal:** Understand relationship between house features and pricing.

---

### 🔹 Task 02: K-Means Clustering – Customer Segmentation
Implemented K-Means clustering to group retail customers based on purchase behavior.

**Applications:**
- Customer segmentation  
- Targeted marketing strategies  

---

### 🔹 Task 03: SVM – Cat vs Dog Image Classification
Built a Support Vector Machine (SVM) model to classify images of cats and dogs using a Kaggle dataset.

**Focus:**
- Image preprocessing  
- Feature extraction  
- Binary classification  

---

### 🔹 Task 04: Hand Gesture Recognition System
Developed a computer vision system to detect and classify hand gestures.

**Applications:**
- Smart devices  
- Accessibility systems  
- Touchless control interfaces
